# ButtonAssist

#### 介绍
ahkv2语言实现的按键宏，鼠标宏，手柄宏，图片识别等功能的工具
#### 环境配置
1.  双击安装Joy目录下的vJoySetup.exe
2.  运行vJoyConf.exe（点击Win键，然后搜索vJoy，点击Configure Joy）
3.  修改界面的Number of Buttons值为32，然后点击apply
4.  双击运行ButtonAssist.exe
